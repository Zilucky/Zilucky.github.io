CVSS

Step 1: Collection of Metrics Sets

Base:

  1. Attack Vector (AV): [N,A,L,P], Network (N), Adjacent (A), Local (L), Physical (P)
  2. Attack Complexity (AC): [L,H], Low (L), High (H)
  3. Privileges Required (PR): [N,L,H], None (N), Low (L), High (H)
  4. User Interaction (UI): [N,R], None (N), Required (R)
  5. Scope (S): [U,C], Unchanged (U), Changed (C)
  6. Confidentiality (C): [H,L,N], High (H), Low (L), None (N)
  7. Integrity (I): [H,L,N], High (H), Low (L), None (N)
  8. Availability (A): [H,L,N], High (H), Low (L), None (N)

Temporal:

  1. Exploit Code Maturity (E): [X,H,F,P,U], Not Defined (X), High (H), Functional (F), Proof-of-Concept (P), Unproven (U)
  2. Remediation Level (RL): [X,U,W,T,O], Not Defined (X), Unavailable (U), Workaround (W), Temporary Fix (T), Official Fix (O)
  3. Report Confidence (RC): [X,C,R,U], Not Defined (X), Confirmed (C), Reasonable (R), Unknown (U)

Environmental:

  1. Confidentiality Requirement (CR): [X,H,M,L], Not Defined (X), High (H), Medium (M), Low (L)
  2. Integrity Requirement (IR): [X,H,M,L], Not Defined (X), High (H), Medium (M), Low (L)
  3. Availability Requirement (AR): [X,H,M,L], Not Defined (X), High (H), Medium (M), Low (L)
  4. Modified Attack Vector (MAV): [X,N,A,L,P], Not Defined (X), Network (N), Adjacent (A), Local (L), Physical (P)
  5. Modified Attack Complexity (MAC): [X,L,H], Not Defined (X), Low (L), High (H)
  6. Modified Privileges Required (MPR): [X,N,L,H], Not Defined (X), None (N), Low (L), High (H)
  7. Modified User Interaction (MUI): [X,N,R], Not Defined (X), None (N), Required (R)
  8. Modified Scope (MS): [X,U,C], Not Defined (X), Unchanged (U), Changed (C)
  9. Modified Confidentiality (MC): [X,N,L,H], Not Defined (X), High (H), Low (L), None (N)
  10. Modified Integrity (MI): [X,N,L,H], Not Defined (X), High (H), Low (L), None (N)
  11. Modified Availability (MA): [X,N,L,H], Not Defined (X), High (H), Low (L), None (N)

Base Metrics Equations

Formula:

ISS = 1 - [ (1 - Confidentiality) × (1 - Integrity) × (1 - Availability) ][1]

Impact[1] =

  • If Scope is Unchanged: 6.42 × ISS
  • If Scope is Changed: 7.52 × (ISS - 0.029) - 3.25 × (ISS - 0.02)^15

Exploitability = 8.22 × AttackVector × AttackComplexity × PrivilegesRequired × UserInteraction

BaseScore[1] =

  • If Impact ≤ 0: 0
  • If Scope is Unchanged: Roundup (Minimum [(Impact + Exploitability), 10])
  • If Scope is Changed: Roundup (Minimum [1.08 × (Impact + Exploitability), 10])

The Specific Scores are as follows:

ISS = 1 - [ (1 - 0.22) × (1 - 0.22) × (1 - 0.56) ] = 0.732304

Impact = Scope is Unchanged = 6.42 × ISS = 4.70139168

Exploitability = 8.22 × 0.85 × 0.77 × 0.62 × 0.62 = 2.06806816

BaseScore = Roundup (Minimum [(Impact + Exploitability), 10]) = Roundup(6.769) = 6.8

Temporal Metrics Equations

Formula:

TemporalScore = Roundup (BaseScore × ExploitCodeMaturity × RemediationLevel × ReportConfidence)[1]

The Specific Scores are as follows:

TemporalScore = Roundup (6.8 × 0.94 × 0.96 × 0.96) = Roundup (5.89) = 5.9
